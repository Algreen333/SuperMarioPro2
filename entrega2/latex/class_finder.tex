\subsubsection*{template$<$typename T$>$\newline
class Finder$<$ T $>$}
La meva implementació de l\textquotesingle{}objecte finder consisteix en un contenidor, dividit en cel·les. Cada element s\textquotesingle{}afegirà a les cel·les que es corresponen amb les coordenades on es troba l\textquotesingle{}objecte.
