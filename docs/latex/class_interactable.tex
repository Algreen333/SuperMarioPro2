\doxysubsection{Detailed Description}
The classs {\ttfamily \doxylink{class_interactable}{Interactable}} is used for every type of object which interacti with \doxylink{class_mario}{Mario}. 

There are 4 types\+:
\begin{DoxyItemize}
\item 0\+: \doxylink{class_coin}{Coin} which is created by a block when mario jumps into it from below. It acts as an animation, floating upwards and disappearing automatically. The coin is added to the coin counter.
\item 1\+: Mushroom which makes \doxylink{class_mario}{Mario} change into its \textquotesingle{}big\textquotesingle{} state. Once it is spawned, it moves horizontally and it is also affected by gravity and block collisitions. It disappears once \doxylink{class_mario}{Mario} touches it.
\item 2\+: The level end flag. It triggers the game win condition once \doxylink{class_mario}{Mario} touches it.
\item 3\+: An enemy \textquotesingle{}goomba\textquotesingle{}. If \doxylink{class_mario}{Mario} jumps on top of it, it is despawned and a coin is spawned. If the goomba touches \doxylink{class_mario}{Mario} in any other way, it damages \doxylink{class_mario}{Mario}. 
\end{DoxyItemize}
